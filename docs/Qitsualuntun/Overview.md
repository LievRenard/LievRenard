### Non-Finite Verb Form

| Infinitive | \-a     |
| ---------- | ------- |
| Supine     | \-#-vat |



### Usage of supine

**Directive**

Dir. supine + kagra → destiny / possiblity

Dir. supine + takagra → obligation



**Locative**

Loc. supine + ua → in progress

Loc. supine + in + ua → begining



**Ablative**

Abl. supine + ua → prohibition
